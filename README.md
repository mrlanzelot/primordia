# Intelligence Planet

A simulation-driven project exploring how intelligence emerges on a planet—from simple organisms to advanced, tool-using species.

## Goal

Simulate evolution where intelligence arises as an emergent property of:
- environment
- embodiment
- memory
- social structures
- culture

No species is predefined to become “human-like”.

## Architecture

The project consists of three layers:

1. Simulation Core
2. Analytics / Explainability
3. UI / Visualization

## First Milestone

“Emergence Explorer”

- Generate a planet
- Simulate evolution
- Inspect species
- Understand why intelligence emerged (or not)

## Tech Direction

- Core: TypeScript (or Python)
- UI: React
- Storage: JSON → SQLite later
